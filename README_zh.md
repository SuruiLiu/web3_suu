## **资源**
我觉得有用的书籍、课程、工具和框架清单:

### **书籍**
- 《精通以太坊》作者: Andreas M. Antonopoulos
- 《区块链基础》作者: Daniel Drescher

### **课程**
- [cyfrin updraft](https://updraft.cyfrin.io/dashboard)

### **工具**
- Foundry: 高性能以太坊开发框架
- Remix: 智能合约在线 IDE
- Alchemy: 强大的 API、SDK 和工具,用于构建和扩展 web3 应用
- Etherscan: 以太坊区块浏览器和分析平台
- Chainlink: Chainlink 定义了链上交易的方式
